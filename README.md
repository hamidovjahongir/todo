# flutter_todo_app

📝 **Flutter To Do App**

A simple and beautiful to-do app built with Flutter. Organize your tasks and boost productivity with a minimal and efficient interface.

📎 Telegramda ko'rish: [Screenshot Link](https://t.me/coderkhamidov/114)

---

## 🚀 Features

- ✅ Add new tasks  
- ✅ Mark tasks as done  
- 🗑️ Delete tasks  
- 📝 Edit tasks
