# flutter_todo_app

📝 **Flutter To Do App**

A simple and beautiful to-do app built with Flutter. Organize your tasks and boost productivity with a minimal and efficient interface.

---

## 🚀 Features

- ✅ Add new tasks
- ✅ Mark tasks as done
- 🗑️ Delete tasks
- 🌙 Light/Dark mode
- 📅 Task date/time (optional)
